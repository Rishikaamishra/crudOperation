# NestJS User Management API

This is a basic User Management API built with NestJS and TypeORM. It includes CRUD operations, pagination, filtering, soft delete, and user restoration.

---

## 🚀 Features

- Create, Read, Update, Delete (Soft delete)
- Pagination & Filtering
- Restore soft-deleted users
- TypeORM + PostgreSQL/MySQL
- DTO validation
